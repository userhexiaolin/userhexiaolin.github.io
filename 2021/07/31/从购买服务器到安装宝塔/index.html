从购买服务器到安装宝塔

七月 31, 2021

一、购买云服务器

1.进入阿里云官网的云服务器购买页面

2.根据需求进行购买（这次选购的是共享性s6<1核2G>;双11搞活动时价格为85元/年）
3.根据短信提示，进入阿里云官网中的控制台，使用提货券
4.输入服务器配置信息（要安装什么操作系统）
5.等待接收包含服务器IP信息的短信

二、安装宝塔

1.在云服务器控制台中放行8888端口（否则后面无法访问宝塔面板）

2.远程连接云服务器
3.通过终端安装宝塔（获得宝塔外网、内网面板地址）

输入命令：（ yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh ）

4.在浏览器输入宝塔面板外网访问地址，登录
5.安装宝塔客户端（解压运行压缩包微信收藏里有）
6.登录宝塔客户端

输入云服务器外网IP

创建登录用户（输入云服务器的登录名名和密码）

选择刚才创建的用户名和密码，登录
